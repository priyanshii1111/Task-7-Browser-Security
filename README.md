# Task 7: Identify and Remove Suspicious Browser Extensions

## Objective
The objective of this task is to learn how to review browser extensions, identify potentially harmful ones, and understand browser security risks.

## Tools Used
- Google Chrome browser
- Screenshot tool (Snipping Tool / Print Screen)
- 
## Steps Performed

1. Opened the browser's extensions manager:  
   - Chrome: `Menu → More Tools → Extensions`

2. Reviewed all installed extensions:  
   - Checked permissions for each extension  
   - Checked reviews and ratings  
   - Determined whether each extension was actually used

3. Identified suspicious or unnecessary extensions.  
   - In this case, both extensions were official and safe:
     - **Google Docs Offline** – Required permissions appropriate, safe, kept.  
     - **McAfee** – Official security extension, required permissions appropriate, kept.  

4. Took screenshots:
   - Screenshot showing all installed extensions before review
   - Screenshot showing permissions/details of extensions

5. Restarted the browser to confirm smooth performance.  

## Observations
- No suspicious or unnecessary extensions were found.  
- Both installed extensions were trusted and safe.  
- Reviewing extensions regularly helps maintain browser security and performance.
